# evolution-of-genetics
This project presents the history of genetics using a GitHub-style interface.

Instead of a traditional report, scientific discoveries are represented as:

- Commits → new ideas and findings
- Branches → competing theories
- Pull Requests → debates and validation
- Issues → unresolved scientific questions


##Concept

Science is not a fixed collection of truths.
It evolves through trial, error, and refinement...much like version control.


##Key Insight

Wrong ideas are not failures — they are necessary commits in the history of truth.


##How to Use

Open index.html in a browser or visit the hosted version
